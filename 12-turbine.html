Turbine


leaf#ss = scale factor - between .9 and 1.1 is suggested


leafJoint# = amount of joints on branch, not too many or crash!


jScale# = scale for branch,requires array with 3 values - [1,2,3]


leafDiv# = number of divisions on the branch



term# = # of leaves minus this number determines if fruit goes here


fruit = boolean - adds balls to the end of branches based on 'term' settings


format is JSON
