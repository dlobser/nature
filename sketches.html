    Associative Arrays

    I built a recursive function that added a heirarchy's elements to an array (with no repeats mind you) and printed that array out. I was then manually copying that print out to define a slew of object variables. Scott Garner showed me how to replace my 40 odd lines of code with one. Associative arrays let you automatically define as many named variables as you want! Shazam

    Updated OBJ exporter

    I found a short snipped of code that exported a given object to the .obj format. For my purposes it was lacking. I added a few lines of code that exported based on world space instead of object and I found a way to export n objects instead of just one (hint: .obj files refer to line numbers when building faces).
